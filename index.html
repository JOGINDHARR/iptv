API Endpoints
All endpoints return JSON. No authentication required.
GET /check Validate a card via real Shopify checkout
ParameterTypeStatusDescription
cardstringRequiredCard in format number|MM|YYYY|cvv
urlstringRequiredShopify store URL (e.g. https://store.myshopify.com)
lowbooleanOptionaltrue = $1–$10 range  |  false = cheapest product. Default: true
proxystringOptionalProxy URL — formats: http://u:p@host:port or host:port:user:pass
Request
GET /check?card=5455122802569146|12|2027|543&url=https://yourstore.myshopify.com&low=true
Response — CHARGED
{
  "card":         "5455122802569146|12|2027|543",
  "status":       "CHARGED",
  "status_code":  "ORDER_PLACED",
  "amount":       "7.49",
  "currency":     "USD",
  "gateway":      "Shopify Payments",
  "site_name":    "yourstore.myshopify.com",
  "receipt_url":  "https://yourstore.myshopify.com/thank-you/...",
  "error":        "",
  "success":      true,
  "checked_at":   "2026-06-23T14:30:01.423"
}

Response Statuses

CHARGED
Real order placed & paid
APPROVED
Card valid (3DS / low funds)
DECLINED

ERROR

Enterprise Features

    
Built for scale with Railway Pro infrastructure.


    

      
🔐
Real TLS Fingerprinting
curl_cffi with randomised browser profiles — Chrome, Edge, Safari. Bypasses bot detection naturally.

      
💳
PCI Card Vaulting
Cards tokenised via Shopify PCI vault (shopifycs.com / shopifyinc.com) before submission.

      
🔗
GraphQL Proposals
Full 5-step checkout proposal flow — email, address, shipping, tax acceptance, signed handles.

      
🌍
Multi-Country Fallback
Automatically tries alternative country addresses when a store doesn't ship to the primary country.

      
♻️
Smart Auto-Retry
Proxy errors trigger a no-proxy retry. Missing cheap variants fallback to cheapest product. Up to 4 attempts.

      
⚡
500 Concurrent Slots
Railway Pro: 24 vCPU, 24 GB RAM. 600-thread pool, no rate limiting, 20 uvicorn workers.

      
📊
Live Receipt Polling
Polls up to 30 times for SuccessfulReceipt, FailedReceipt, or ActionRequiredReceipt (3DS).

      
🛡
Tax Change Handling
Detects TAX_NEW_TAX_MUST_BE_ACCEPTED and retries submission with updated amounts automatically.

      
🌐
Always Shopify Payments
Every check goes through Shopify's native payment gateway — no third-party processors involved.
